    Hello World!


    This is an simple editable area.


    1. Select a text to reveal the toolbar.
    2. Edit rich document on-the-fly, so elastic!


    End of simple area

    Please include following css file at head element

    <!-- Quill css -->
<link href="assets/vendor/quill/quill.core.css" rel="stylesheet" type="text/css" />

<!-- quill js -->
<script src="assets/vendor/quill/quill.js"></script>
<!-- quill Init js-->
<script src="assets/js/pages/demo.quilljs.js"></script>
